Theme Customization Report

Store: growmodo-test  •  Developer: Jim Joco  •  Date: June 2026

Overview

This document summarizes the work completed on the Growmodo Test Shopify store. The goal was to build a functional, visually polished storefront based on the provided design — covering everything from store content setup to full theme customization.

Technologies Used

  • Shopify CLI — used to run a live local preview of the theme during development, so changes could be reviewed before going live
  • Liquid — Shopify's templating language, used to modify how the header, search, and other theme components are structured and rendered
  • SCSS — a powerful stylesheet language used to write all custom styles in a clean, organized, and maintainable way
  • Git & GitHub — version control and deployment; the live theme only updates when code is pushed to the repository
  • Claude AI — used as a development assistant to help implement and refine theme customizations efficiently

Development Approach

A safe, professional workflow was used throughout the project. Changes were first previewed on a private development theme using Shopify CLI before being pushed to the live theme through GitHub. This ensured the live store was never accidentally broken during development.

Changes were always tested locally first → reviewed on the dev theme → then deployed to the live theme via GitHub. The live store only updates when code is intentionally pushed.

Store Content Setup

Before any design work, the store was populated with real content to work with:

  • Created 9 sample products with titles, descriptions, pricing, and images
  • Organized products into collections to reflect the store's category structure
  • Set up and modified navigation menus — including the main header menu and any supporting links — to match the intended site structure
  • Did an initial product page setup, ensuring the layout and key details (price, description, add to cart) display correctly

Custom Fonts

Two custom font families were uploaded and integrated into the theme — Integral CF for headings and display text, and Satoshi for body copy and supporting text. These replace the default theme fonts and are applied consistently across all sections of the store.

Header

The header was significantly reworked to match the design:

  • Replaced the default popup search window with a permanent inline search bar that sits directly in the header, similar to how major retail sites present search
  • The search bar only shows suggestions when a customer is actively typing — keeping the header clean otherwise
  • Swapped the default icons for custom branded icons for search, cart, and account
  • Adjusted the order and spacing of header elements to match the design layout
  • Styled the navigation links and logo to use the correct custom fonts

Homepage Sections

Each section of the homepage was styled to match the design:

  • Hero Banner — set the correct height, text layout, font styles, and a pill-shaped call-to-action button
  • Product List — styled the section heading, product names, and pricing. Removed unnecessary badge labels and styled the "View All" button
  • Collection List — wrapped the section in a rounded grey card to create a visually distinct area, with matching fonts and sizing
  • Dividers — softened the divider lines between sections for a cleaner look
  • Email Signup — applied heading font and fixed a stacking issue so the section displays correctly over other content

Footer

The footer was styled to overlap slightly with the section above it, creating a layered visual effect that matches the design. Font styles were applied to the logo text, description, menu headings, and menu links.

Product Page

An initial setup was done on the product page to ensure it displays the essential information correctly — including product title, price, description, and the add-to-cart button — in preparation for further refinement.
